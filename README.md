# 🚀 NUIST 计算机科学学习资源库 (Resources-NUIST)

> **“浦口的风再大，也吹不走我们要卷代码的决心。”**

欢迎来到 **Resources-NUIST**。这是一个由南信大学子共同维护、为南信大学子服务的计算机及相关专业学习资料合集。本项目旨在打破信息差，整理从大一到大四、从基础课到专业课的核心资料，助你在代码与算法的海洋中顺利“上岸”。

---

## 📂 资源概览

本库目前涵盖了计算机学院（计院）、人工智能学院、软件学院等相关专业的重点课程。我们将资源划分为以下几个核心板块：

### 1. 📚 核心课程图谱

* **公共基础课**：高数、大物、线性代数、离散数学（拿高绩点的敲门砖）。
* **计算机基础**：C/C++程序设计、数据结构与算法、计算机组成原理、操作系统、计算机网络。
* **专业选修/进阶**：数据库系统、人工智能导论、机器学习、编译原理、网络安全、软件工程。

### 2. 📝 历年复习宝典

* **往年卷/模拟题**：整理了学长学姐贡献的历年真题及期末模拟卷。
* **高分笔记**：针对课程难点（如计组的指令集、操作系统的进程调度）的详细解析。
* **考试重点回顾**：由“过来人”总结的期末考试避雷指南。

### 3. 💻 实验与项目实战

* **课程实验**：实验报告模板及参考代码（拒绝直接Copy，重在理解思路）。
* **大作业展示**：包含Web开发、移动端APP、小型编译器等优秀课设展示。
* **竞赛资料**：蓝桥杯、ACM、数学建模及互联网+等赛事的备赛资源。

### 4. 🎓 升学与就业

* **考研资料**：本校及名校计科专业考研历年报录比、专业课经验分享。
* **保研手册**：夏令营、预推免的申请流程及面试常见算法题。
* **就业求职**：大厂面试经、简历模板、计算机领域职业规划建议。

## 🛠 如何参与贡献？

本项目秉持 **“前人栽树，后人乘凉”** 的开源精神。如果你手里有更有价值的资料，欢迎参与贡献：

1. **Fork** 本仓库。
2. 新建一个分支：`git checkout -b feature/your-contribution`。
3. 上传你的资料（建议将文档整理为 PDF 或 Markdown 格式）。
4. 提交 **Pull Request**。

> **小贴士**：请在上传前检查是否涉及版权敏感内容，保护隐私，尊重学术诚信。

## ⚠️ 声明

1. **仅限学术交流**：本仓库所有资料仅供个人学习和研究使用，严禁用于任何商业用途。
2. **学术诚信**：参考实验代码是为了帮助理解，请勿直接将其作为作业提交。学术诚信是程序员的底线！
3. **反馈意见**：如果发现资料有误或存在侵权内容，请立即通过 [Issue](https://www.google.com/search?q=../../issues) 告知，我们会第一时间处理。
